# Teknoloji Yığını Farkındalığı

1. **Önce Analiz Et:** Kod yazmadan önce `package.json`, `requirements.txt` veya `go.mod` dosyasını oku.
2. **Tutarlılık:** Projede zaten kurulu bir kütüphane varsa (örn: `date-fns`), aynısını yapacak yeni bir kütüphane (örn: `moment.js`) önerme/kullanma.
3. **Versiyon Uyumu:** React 18 kullanılıyorsa, React 16 class component syntax'ı kullanma.
